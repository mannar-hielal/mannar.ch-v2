eDuca - Course Landing page

  • Web_Development

eDuca landing page is a light-weight landing page for a course, a workshop or a seminar, boost your sales by presenting a demo video of your course/workshop uploaded on youtube or vimeo.

With attractive gradient colors, eDuca was develop not only load fast but also to set up a high-converting sales by getting visitors to use the contact form.

Hours were put on this landing page to make it easy-to-customize, faster to reload & error free.

Features

  • "Watch Demo" video in the hero section (youtube or vimeo)
  • Bootstrap 5
  • Fully Responsive
  • Portfolio with Filtering
  • A functioning contact-us form with validations
  • Google analytics integration
  • Favicon, pinned icon & sitemap.
  • 404 page & thank you page.
  • Easy-to-customize fonts, colors, photos to match your branding.
  • one-line compiler: npm run compile:sass

What's included

eDuca web theme includes the following files:

  1. SASS files grouped by sections, components, making it easy to find what you need to edit.
  2. Images folder that hold all the images used, grouped in folders.
  3. Assets folder that holds the CSS, JS & fonts.
  4. Index Page Template prepared with Google analytics Template script.
  5. Thank you Page Template.
  6. Favicon & facicon set for different devices.
  7. Sitemap.xml for SEO purpeses. ells search engine crawlers which URLs the crawler can access on your site.
  8. Robots.txt which tells search engine crawlers which URLs the crawler can access on your site.
  9. .htaccess for caching assets & boosting page load.
